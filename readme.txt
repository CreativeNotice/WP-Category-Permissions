Provides a simple interface for managing viewability on the front end, and access to post to a category on the backend.

== Description ==

Using this plugin you may limit access to any given category by attaching permissions via Wordpress Roles.

== Installation ==

The plugin is simple to install:

1. Download 'WP-Category-Permissions.zip'
1. Unzip
1. Upload 'WP-Category-Permissions.zip' directory to your '/wp-content/plugins' directory
1. Go to the plugin management page and enable the plugin
